# Student Management System

A web-based Student Management System developed using **C++**, **Crow Framework**, **SQLite**, and **CMake**.

## Features

- Admin Login
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Responsive UI
- SQLite Database

## Technologies Used

- C++
- Crow Framework
- SQLite
- HTML
- CSS
- JavaScript
- CMake

## Project Structure

```
StudentManagementSystem/
│── include/
│── src/
│── templates/
│── static/
│── build/
│── CMakeLists.txt
```

## How to Run

```bash
mkdir build
cd build
cmake ..
cmake --build .
./StudentManagementSystem
```
